API

The following table lists the interfaces to manage CiG resources. It lists the resource name, its URI, the HTTP method, its parameters, and the response from the API to the method.

Resource Method Resource ID Description Returns Parameters Examples Experiment GET /exp Experiment details List of result
  • size (optional): Number of vertices in the randomly generated graph.
  • cycles (optional): Should the random graph be generated with cycles? true or false
  • #{restRoot}/exp
  • #{restRoot}/exp?size=1000&cycles=false
  • #{restRoot}/exp?size=5000&cycles=true
